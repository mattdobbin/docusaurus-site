### Ellipsis Meetings Dialog: Layout and Components

The Ellipsis layout and components for the Ellipsis Meetings dialog are AFR attributes:
:::info
**AFR Path**: ACE.Core.QuickAdd.Layouts.QAM
:::

| Attribute | Description | Supported Values | Default |
|-----------|-------------|------------------|---------|
| `BodyLayout` | Defines the layout for the inner body of the dialog. | Ellipsis Configuration | n/a |
| `Buttons` | Defines the buttons on the dialog. | Ellipsis Configuration | n/a |
| `DialogClass` | Holds any additional classes to append to the dialog. | Ellipsis Configuration | n/a |
| `DialogTitle` | Defines the title of the dialog. | STRING | Create Meeting |
| `DragHandle` | Configure the handle to allow dragging of the dialog. | Ellipsis Configuration | .dialog-header |
| `Layout` | Defines the outer layout of the dialog. | Ellipsis Configuration | n/a |
| `LayoutLoaderOptions` | Defines the options for the layout loader to ensure it merges and loads a standard template layout. | Ellipsis Configuration | n/a |
| `ValidationRules` | Defines the Validation Rules that are applied to the dialog. | Ellipsis Configuration | n/a |

:::warning
The Tier1 Ellipsis Framework is under development, and is currently in the beta version. This framework requires significant development knowledge to use. Contact Tier1 Customer Success for further information.
:::

### Launch Points
The Ellipsis Meetings dialog can be launched from the Main toolbar and Contact Context menus. The launch points can be configured with the following AFR attributes:

:::info
**AFR Path**: ACE.ALM.ApplicationSettings.Toolbar.ActionButtons.Contact.QAM
:::

| Attribute | Description | Supported Values | Default |
|-----------|-------------|------------------|---------|
| `Data` | The special key used to determine the action that occurs when the icon in the Main toolbar is clicked. | URL or key used to determine the action. To configure the dialog to launch as a standalone window, add **LaunchType** to the end of the URL. Example: **ShowEllipsisMeetingsForm?layoutPath= ACE.Core.QuickAdd.Layouts. QAM&launchType=window** | ShowEllipsisMeetingsForm |
| `Icon` | Defines the icon that is displayed for the button. | Icon Name | n/a |
| `Label` | Defines the label text that appears when users hover over the icon. | STRING | Create Meeting |
